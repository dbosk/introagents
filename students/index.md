# Getting started with agentic LLM use

This section collects practical information for students who want to get started
with agentic use of LLMs in a way that fits this course and the surrounding
tooling.

The main path we recommend is:

1. Create a GitHub account.
2. Apply for GitHub Education as a student.
3. Enable GitHub tools that come with student benefits, especially Copilot.
4. Move on to command-line workflows such as GitHub Copilot CLI (`copilot`),
   `opencode`, and the Python package `llm`.

## Why start with GitHub?

GitHub is the natural first step because GitHub Education gives eligible
students access to GitHub student benefits, including the student Copilot plan.
That makes it the easiest way to begin using agentic AI tools without paying for
an individual subscription first.

## Tool guides

These pages cover:

- GitHub Copilot CLI (`copilot`) for the simplest path from GitHub student
  benefits
- `opencode` for a more capable open source coding agent
- the Python package `llm` for shell pipelines, scripts, and Python programs
- a comparison page for choosing the right tool for the task
